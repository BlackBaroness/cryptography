Ultimate encoding and hashing library, 
based mainly on [norkator/cryptography](https://github.com/norkator/cryptography).

### Supported hashes:

- Adler32
- Blake2b (160, 256, 384, 512)
- CRC (8, 16, 24, 32, 64)
- MD (2, 4, 5)
- SHA (0, 1, 224, 256, 384, 512)
